About and URL instructions

What this does

Builds an Excel ride schedule for only the riders you select. Start with the show URL, load riders, choose your barn riders, load their classes, then generate the Excel file.

Which URL to paste

  • HorseShowOffice: open Ride Times Lookup from the show page and paste a URL like https://www.horseshowoffice.com/hso/ridetimes.asp?s=5298&o=50
  • FoxVillage: paste the show page URL like https://www.foxvillage.com/show?id=11741
  • Equestrian Hub: paste the show page URL like https://equestrian-hub.com/show/274044

Arena and ring info

HorseShowOffice usually needs the Ride Schedule PDF URL to fill arena names. If the show has only one ring, choose One-ring show instead.
